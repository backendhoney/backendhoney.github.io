공정한 평가를 위한 그림 커뮤니티 안드로이드 애플리케이션

• 자바 언어로 개발된 고급 그림 커뮤니티 애플리케이션
• Amazon Web Service 인스턴스를 생성한 후, Flask 라이브러리로 파이썬 서버 구축
• 우분투 리눅스에 MySQL을 넣고 PyMySQL로 플라스크와 연동해서 RestFul API 서버에 적재
• 안드로이드에서는 Retrofit을 이용하여 HTTP 기반 통신으로 서버 값을 받아 온다.
• 앱바, 탭 레이아웃, 뷰페이저, 리사이클러뷰, 코디네이터 레이아웃, 스태거드 그리드 레이아웃, 팝업 메뉴
• UI는 사용자가 한눈에 알아볼 수 있도록 가능한 직관적으로 설계했고 터치 빈도가 높은 위젯의 경우 아래쪽에 둠

본 프로젝트는 소프트웨어 공학의 Plan-driven, 폭포수 모델을 따라서 개발이 진행되었습니다.
